HyperparameterSpace API

üìã Overview

Define the search space for hyperparameters and assign them to different agents for specialized optimization.

üèóÔ∏è Initialization

python 
# Method 1: Dynamic addition
space = HyperparameterSpace()
space.add_continuous('learning_rate', 1e-4, 1e-2, agent=0)

# Method 2: Full initialization
space = HyperparameterSpace(
    agent0_params=['learning_rate', 'weight_decay'],
    agent1_params=['hidden_size'],
    agent2_params=['batch_size'],
    bounds={'learning_rate': (1e-4, 1e-2), ...},
    param_types={'learning_rate': float, ...}
)

üîß Methods

add_continuous(name, min_val, max_val, agent)

add_continuous(name: str, min_val: float, max_val: float, agent: int) -> self

Add a continuous hyperparameter with min/max bounds.

python 
# Float parameters
space.add_continuous('learning_rate', 1e-5, 1e-2, agent=0)
space.add_continuous('dropout_rate', 0.0, 0.5, agent=1)

# Int parameters (will be cast from continuous range)
space.add_continuous('hidden_size', 64, 512, agent=2)
space.add_continuous('batch_size', 16, 128, agent=2)

add_discrete(name, choices, agent)

add_discrete(name: str, choices: List, agent: int) -> self

Add a discrete hyperparameter with fixed choices (use sparingly - continuous ranges are preferred).

python 
space.add_discrete('optimizer', ['adam', 'sgd', 'adamw'], agent=0)
space.add_discrete('activation', ['relu', 'tanh', 'gelu'], agent=1)

üéØ Agent Assignment

Agent 0

Problem-specific parameters like learning rates, regularization, class weights

Agent 1

Model architecture parameters like hidden layers, network size, activation functions

Agent 2

Training parameters like batch size, dropout rate, optimization settings

üìù Complete Example

python 
from MAT_HPO_LIB import HyperparameterSpace

# Create space for neural network optimization
space = HyperparameterSpace()

# Agent 0: Problem-specific parameters
space.add_continuous('learning_rate', 1e-5, 1e-2, agent=0)
space.add_continuous('weight_decay', 1e-6, 1e-3, agent=0)

# Agent 1: Architecture parameters  
space.add_continuous('hidden_size', 64, 512, agent=1)
space.add_continuous('num_layers', 2, 8, agent=1)

# Agent 2: Training parameters
space.add_continuous('batch_size', 16, 128, agent=2)
space.add_continuous('dropout_rate', 0.0, 0.5, agent=2)

# Each agent will have 2 parameters to optimize
print(f"Agent dimensions: {space.agent_dims}")  # (2, 2, 2)
