MAT-HPO Library

Multi-Agent Transformer Hyperparameter Optimization for complex machine learning problems. Leverage the power of reinforcement learning and transformer networks to find optimal hyperparameters automatically.

Get Started View on GitHub

üöÄ Key Features

ü§ñ

Multi-Agent Architecture

Three specialized agents optimize different types of hyperparameters simultaneously, enabling efficient parallel search across the parameter space.

üß†

Transformer Networks

Uses advanced transformer neural networks for intelligent hyperparameter selection, learning from optimization history to make smarter choices.

üîß

Flexible Integration

Easy-to-use abstract interface that works with any ML model or training pipeline. Simply inherit from BaseEnvironment and implement four methods.

‚ö°

CUDA Acceleration

Full GPU support for faster optimization. Automatic memory management and efficient tensor operations for large-scale hyperparameter search.

üìä

Comprehensive Logging

Detailed optimization history tracking, performance metrics, and result analysis. Built-in support for experiment management and reproducibility.

üéØ

Advanced Features

Multi-objective optimization, constraint handling, early stopping, checkpointing, and custom callback systems for complex optimization scenarios.

üèóÔ∏è Architecture Overview

üéØ

Agent Specialization

  • Agent 0: Problem-specific parameters (class weights, regularization)
  • Agent 1: Model architecture parameters (hidden size, layers)
  • Agent 2: Training parameters (batch size, learning rate)
üîÑ

Optimization Loop

  1. Agents propose hyperparameters
  2. Environment evaluates configuration
  3. Shapley value-based credit assignment
  4. Agents learn and improve strategies
  5. Repeat until convergence
